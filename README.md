# Cybersecurity Champion Starter Toolkit

This toolkit is designed for system administrators transitioning into the Cybersecurity Champion role.
It provides ready-to-use scripts for auditing Windows, Ubuntu, and database systems, along with reporting templates.

## Folder Structure
- **windows-scripts/** – PowerShell scripts for AD, local admins, password audits.
- **ubuntu-scripts/** – Shell scripts for system audit, patch checks, sudoers review.
- **database-scripts/** – SQL scripts for Oracle and SQL Server user and privilege audits.
- **report-templates/** – Excel and CSV templates for reports and dashboards.

## Usage Guidelines
1. Run scripts in a test environment first.
2. Schedule recurring audits using Task Scheduler (Windows) or cron (Linux).
3. Export results to the report templates.
4. Document findings and recommendations.
